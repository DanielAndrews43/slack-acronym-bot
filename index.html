Features

Define Words

Quickly define words by using our /explain command followed by the word and the definition.

Explain words

Can't find the definition of a word? Find the definition by typing /explain followed by the word.

Crowdsourced

If a word hasn't been defined yet, the bot will quickly find answers by posting to a public channel.
